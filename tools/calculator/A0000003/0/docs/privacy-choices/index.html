Privacy Choices

Simple ways to manage privacy in the current Great Calculator release.

Great Calculator is designed so that most privacy controls are direct and local. This page explains the practical choices available in the current release.

Current release

Effective date: March 24, 2026

Applies to the current iPhone and iPad release of Great Calculator.

Questions?

trentshurts@gmail.com

1. Clear saved history

Use the app’s history controls to remove individual history items or clear all saved history. This removes committed expressions and results stored for recall inside the app.

2. Avoid live currency-rate requests

If you prefer not to send a one-time exchange-rate request, do not use the live currency conversion lookup. Most other calculator features continue to work fully on-device.

3. Choose whether to send support email

The app uses a standard mail link for support. You decide whether to open your mail app and whether to send a message or attachments.

4. Remove local app data

Uninstalling the app removes the app and its sandboxed local storage from your device, including settings, history, and the local rate cache.

No account deletion workflow is needed

The current release does not create or require user accounts.

Because the app does not provide account creation or sign-in, there is no in-app account deletion flow for the current release. If that changes in a future release, the app and this site should be updated before release.

Current release posture: no account, no cloud profile, and no cross-app tracking.

What this page does and does not change

  • It helps you understand your options: clear history, avoid live requests, avoid support email, or remove the app.
  • It does not create new data flows: visiting this page does not change how the app itself processes data.
  • It does not add tracking: the current app release does not enable Apple Ads attribution and does not include third-party analytics SDKs.

Need help with a privacy question?

If you have a privacy question about Great Calculator, contact trentshurts@gmail.com.
