What makes this site app-specific

The content here is tailored to the current Great Calculator codebase rather than a generic privacy-policy template.

Local-first behavior

The policy reflects the app’s local storage of preferences, onboarding acceptance flags, calculation history, and a small exchange-rate cache inside the app’s private sandbox.

User-initiated network activity

The policy specifically documents the Money → Currency live-rate lookup path and the optional support-email path, instead of making broad or inaccurate claims.

Disabled Apple Ads attribution path

The current release contains a default-off Apple Ads attribution integration, but the app metadata flag keeps it disabled. No Apple Ads attribution request is made in the current release.

No HealthKit integration

The app includes calculator features that can compute health-related formulas, but the current release does not read or write Apple Health or HealthKit data.

Hosting note: If this site is hosted on GitHub Pages, GitHub states that visitor IP addresses are logged and stored for security purposes. That behavior is about the website host, not the app itself.
