Accessibility

Accessibility information for Great Calculator.

Great Calculator is intended to be usable on iPhone and iPad with clear controls, readable presentation, and a support path for accessibility feedback.

Feedback

trentshurts@gmail.com

Important note

Accessibility quality should always be verified on supported devices, text sizes, and OS versions.

Current accessibility posture

The current release is designed with accessibility-conscious interaction patterns and UI structure.

Control clarity

Important controls are designed to be clearly labeled and visually distinct across the main calculator flows and menu surfaces.

Appearance support

The app supports light and dark appearances to help preserve contrast across common usage environments.

Readable sizing

The interface is built with scalable typography and adaptive layout behavior for phones and tablets.

Touch-first interaction

Key calculator and menu interactions are designed around generous touch targets and structured visual grouping to reduce ambiguity.

Best way to report an accessibility issue

  1. Describe the device model and iOS/iPadOS version.
  2. Identify the calculator type or screen where the issue occurred.
  3. List the accessibility feature in use, such as VoiceOver or larger text.
  4. Describe the exact problem and the expected behavior.

Email accessibility feedback to: trentshurts@gmail.com
