Privacy questions
You can use the same support email for privacy questions or data-related requests.

Before sending a report: Please include the calculator type you used, the exact expression or inputs, what you expected to happen, what happened instead, and the app version/build if available.

Quick troubleshooting

  • Tap = to commit a result before expecting it to appear in history.
  • If a preference seems stuck, relaunch the app after changing the setting.
  • For Money conversions, confirm that the device has network access if a live rate is required.
  • For equation workflows, include the exact expression you entered in the report.

Privacy-related requests

  • Review the Privacy Policy for current-release data handling details.
  • Use the Privacy Choices page for direct local-data management guidance.
  • If you need to ask a privacy question, send it to the support email above.
