Date tracker, countdowns, and milestones organized around time.

Project Milepost helps you track personal dates, countdowns, anniversaries, birthdays, deadlines, goals, memories, and bundled historical milestones as cards, a grid, or a visual timeline.

Release summary

Free download

Project Milepost is intended to be listed as a free app.

Advertising

Google AdMob banner advertising is included. Ads are hidden after the one-time Remove Ads purchase is confirmed by Apple.

Remove Ads

The optional one-time Remove Ads purchase is intended to be $0.99 in the United States.

Bundled history

The 20 curated historical events are included with the app and are not sold separately.

Optional Calendar import

Calendar import starts only when you tap the Calendar import button and choose events to copy into Project Milepost.

Optional local photos

You may choose event background photos. Selected photo copies are stored locally for the app.

No Project Milepost account

The app does not require sign-in, does not run a Project Milepost user account system, and does not maintain a Project Milepost server for your personal events.

Email-only support

Support is available at trentshurts@gmail.com. No phone number is listed for Project Milepost support.
