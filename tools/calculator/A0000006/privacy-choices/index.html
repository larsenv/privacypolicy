Privacy Choices

Current controls for Great Calculator data.

Great Calculator is local-first. These are the practical choices available in the current iPhone and iPad release.

Read the Privacy PolicyEmail privacy support

Local history

Clear saved calculation history inside the app when you no longer want previous expressions and results available for recall.

Preferences

Change theme, number formatting, answer format, percent mode, and haptics inside the app. These preferences are stored locally.

Currency rates

Live currency-rate lookup is optional. Avoid using the live rate action if you do not want that request to be made.

Support email

Support email is voluntary. You control what you write, whether you attach screenshots, and whether you send the message.

Remove local data

Uninstalling the app removes app-local data from the device, subject to ordinary operating-system behavior.

Questions

Contact support for privacy questions that are not handled by an in-app control.

Important context

The app does not require an account, does not include third-party analytics SDKs, does not include advertising SDKs, and does not track users across apps or websites in the current release.

Visiting this public website is separate from using the app. Website hosts can process standard server logs for security and operations.
