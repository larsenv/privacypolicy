What to include

The best reports identify the exact screen, event data, expected behavior, and actual behavior.

Useful report details

  • Device model and iOS or iPadOS version.
  • App version and build if available.
  • Screen: card, grid, timeline, event editor, Calendar import, Ads, Remove Ads, History Packs, or settings.
  • Event precision: date-only or date-and-time.
  • Whether the issue involves Calendar import, photos, ads, purchase restore, or history events.
  • Steps to reproduce the behavior.
