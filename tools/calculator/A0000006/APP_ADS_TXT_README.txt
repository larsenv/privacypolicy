# app-ads.txt setup for Project Milepost

Project Milepost uses Google AdMob compact top banner advertising in Build 40.

Before public release:

1. Open Google AdMob.
2. Copy the real Google publisher value.
3. Replace `pub-REPLACE_WITH_YOUR_GOOGLE_PUBLISHER_VALUE` in `app-ads.txt.TEMPLATE`.
4. Save the completed file as `app-ads.txt`.
5. Deploy `app-ads.txt` at the root level of the public developer website domain that appears on the App Store listing.

Do not publish the placeholder value.
