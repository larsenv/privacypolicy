Accessibility

Accessibility information for Project Milepost.

Project Milepost is intended to be usable on iPhone and iPad with readable layout, touch-friendly controls, light and dark appearance support, and direct accessibility feedback.

Email Accessibility FeedbackSupport
Readable layoutAdaptive interfaceTouch-first controlsFeedback welcome

App accessibility posture

The app uses structured controls, grouped sheets, readable typography, and adaptive layout behavior for card, grid, timeline, Calendar import, ads, and event editor flows.

Appearance and motion

The current release supports light and dark appearances and uses focused interface motion rather than requiring rapid or complex gestures.

Website accessibility

This website uses semantic headings, visible links, skip navigation, keyboard focus styles, responsive layouts, and plain-language support paths.

Report an issue

Email trentshurts@gmail.com with the device, system version, screen, accessibility setting, expected behavior, and actual behavior.
