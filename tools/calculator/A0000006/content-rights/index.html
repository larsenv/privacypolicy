Content Rights

Public content-rights information for Great Calculator.

Great Calculator is primarily an on-device calculator and utility app. This page explains the app’s public third-party content path for optional exchange-rate data.

SupportPrivacy Policy

Content-rights posture

This page is limited to the app’s actual public third-party content path: optional exchange-rate data used for user-requested currency conversion.

Summary

Primarily on-device, with one optional third-party data path

What the app mainly uses

Great Calculator is primarily an on-device utility app for math, graphing, geometry, time, conversion, finance, counting, and number-analysis workflows.

Third-party content path

The current release can optionally access exchange-rate data for the Currency workflow in the Money calculator when the user explicitly requests a rate lookup.

What the app does not include

The app does not include user-uploaded media, music streaming, video streaming, news feeds, celebrity content, or social-content catalogs.

User control

Most app features work locally. The user decides whether to use the live currency conversion path that triggers a one-time rate request.

Exchange-rate provider

Public explanation of the provider relationship

The app identifies Frankfurter as the exchange-rate provider for the optional Currency workflow. The provider is used to retrieve exchange-rate data needed to display user-requested conversion results.

  • Provider: Frankfurter (api.frankfurter.dev)
  • Use: user-requested exchange-rate lookup for currency conversion
  • Mode: optional and on demand
  • Processing: the amount calculation is handled in the app after the rate is returned

Questions about privacy, support, or public disclosure can be directed to the support page for Great Calculator.

Why this page exists

This page provides a stable public explanation of the app’s third-party content path and supports the Content Rights information maintained in App Store Connect.
