Age Suitability

Age suitability for Project Milepost.

Project Milepost is a general-purpose date tracker for personal milestones, countdowns, Calendar events a user chooses to import, local event photos, and bundled historical milestones.

Plain-language suitability summary

Project Milepost is designed for a broad audience, including families, students, adults, and older children who are using the app with appropriate parent or guardian oversight. It is not designed or marketed as a children-only app.

The app may show Google AdMob banner advertising unless the optional Remove Ads purchase is active. Because advertising is included, the app should not be represented as child-directed unless the developer has separately completed every child-directed-app advertising, privacy, and platform requirement.

Current build review

Content and capability checklist.

This checklist reflects Project Milepost version 1.2, build 31, based on the current public release plan.

Personal dates and timelines

The main function is creating and viewing date-based event records, countdowns, elapsed-time entries, memories, goals, deadlines, anniversaries, birthdays, and historical milestones.

Curated historical content

The bundled historical milestones are preselected and included in the app. The app does not generate new public history content inside the app.

No social features

The app does not include public posting, public profiles, user-to-user messaging, social networking, or user-generated public content.

No unsafe-content categories

The app is not intended to include gambling, contests, loot boxes, weapons instructions, drug or tobacco content, sexual content, graphic violence, horror content, or mature entertainment content.

No sensitive advice role

The app is not a medical, treatment, financial, legal, emergency, safety, or professional-advice product.

Optional permissions

Calendar import and local photo selection are optional. The app should ask only when the user starts the related feature.

Advertising

Google AdMob banner advertising may appear near the bottom of the main app screen unless Remove Ads is active.

Optional purchase

Remove Ads is planned as a one-time optional purchase handled by Apple. It should hide banner ads when confirmed.

Age rating submission guidance

Apple determines the App Store age rating from the age rating questionnaire in App Store Connect. The developer should answer that questionnaire truthfully based on the exact app build submitted for review.

For the current Project Milepost build plan, the expected owner answers should reflect: no social networking, no public user-generated content, no unrestricted web browser, no gambling, no contests, no loot boxes, no medical or treatment guidance, no precise location sharing, no account requirement, optional Calendar access, optional photo selection, Google AdMob banner advertising, and an optional Apple-handled Remove Ads purchase.

This page is not a substitute for the final App Store Connect questionnaire. If any feature changes before submission, especially advertising, web links, historical content, content generation, user sharing, or external content, this page and the age rating answers should be reviewed again before release.

Parent and guardian notes

Project Milepost can be used to track family dates, school deadlines, events, personal goals, and memories. A parent or guardian can manage app access, purchases, and advertising limits through Apple device settings and the App Store account settings used on the device.

Personal event records are intended to stay on the device unless the user chooses to share information outside the app using operating-system features. Support requests are handled by email only.
