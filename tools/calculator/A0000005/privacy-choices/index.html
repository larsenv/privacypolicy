Privacy Choices

Manage Project Countdown privacy choices.

These options describe how to manage local countdown data, optional photos, support email, and permissions for the current release.

Read Privacy PolicyContact Support

Delete countdown events

Delete individual events from within the app. Removing an event removes that event’s title, date, notes, precision settings, and related local metadata.

Remove custom photos

Clear a selected event background photo from the event editor or delete the event using that photo.

Revoke photo access

Use iOS Settings to change or revoke Project Countdown photo-library permission.

Disable history packs

Turn off curated history packs in the app to remove those generated history events from your local event list.

Delete app data

Delete the app from the device to remove its local sandbox data, including countdowns, preferences, local photos, and recovery copies.

Choose what to email

Support email is optional. Include only the details you want to share.
