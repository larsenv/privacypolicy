What to include

The best reports identify the exact screen, event data, expected behavior, and actual behavior.

Useful report details

  • Device model and iOS / iPadOS version.
  • App version and build if available.
  • View mode: card, grid, timeline, composer, widget, or settings.
  • Event precision: date-only or date-and-time.
  • Device time zone and whether the issue appears after a time-zone change.
  • Steps to reproduce the behavior.
