Age Suitability

General productivity use with user-created countdowns.

Project Countdown is designed as a date, milestone, and timeline utility. Final App Store age-rating determinations should follow App Store Connect’s questionnaire for the submitted build.

User-created titles and notes

Users can create their own event titles and optional notes. The app does not provide public sharing or user-generated public feeds in the current release.

Optional photos

Users may select local photos as event backgrounds. The selected content is controlled by the user and stored locally in the app sandbox.

History packs

Curated historical milestones may appear if the user enables history packs. These should be reviewed against final age-rating metadata before submission.
