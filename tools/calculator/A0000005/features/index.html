Features

Purpose-built countdown tools for personal events and historical milestones.

Project Countdown combines fast event creation, view-mode flexibility, optional personalization, curated history packs, and widget support.

Core tools

Countdown features included in the current app.

Personal events

  • Create, edit, and delete countdown events.
  • Add optional notes for context or memory keeping.
  • Support date-only and date-and-time event precision.

Adaptive views

  • Card view for focused countdowns.
  • Grid view for scanning many events.
  • Interactive timeline view for chronological exploration.

Timeline engine

  • SpriteKit-backed panning and zooming.
  • Clustered event layout for dense timelines.
  • Focused event navigation from app state and widgets.

History packs

  • Curated milestone packs can be enabled or disabled.
  • History events sync locally with the event store.
  • History-pack choices stay in local preferences.

Optional photos

  • Choose a local photo for an event background.
  • Photo-library access is requested only when needed.
  • Selected images are stored as app-optimized local copies.

Widgets

  • Widget snapshots support featured countdown presentation.
  • Deep links can route back into relevant app views.
  • Shared widget support is scoped to the app group configuration.

Compatibility note

Project Countdown is designed for iPhone and iPad. The App Store listing and the submitted build remain the source of truth for exact platform availability, minimum OS version, and compatibility details.

Local-first posture

Core countdown data and preferences are stored locally. No account is required for the current core experience.
