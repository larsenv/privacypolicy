What sets it apart

Personal countdowns with clear privacy boundaries.

The current app focuses on meaningful date tracking, local persistence, optional personalization, and public support resources.

Create and manage countdowns

Save personal events with titles, optional notes, date-only or date-and-time precision, and custom presentation choices.

Card, grid, and timeline views

Switch between focused event cards, a practical grid, and a SpriteKit-powered timeline for chronological exploration.

Optional custom photos

Use a selected photo as an event background only when you choose to grant photo-library access.

Local-first storage

Countdown data, notes, view preferences, history-pack choices, and local backup recovery data stay inside the app sandbox by default.
