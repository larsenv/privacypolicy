Privacy Policy

Project Countdown is local-first by design.

This policy describes the current Project Countdown release for iPhone and iPad. The core app does not require an account and does not track users across apps or websites.

What the app stores locally

The following information is used to provide the countdown experience and remains in the app’s local storage unless you delete it or remove the app.

Countdown events

Event titles, dates, date precision, optional time values, notes, and view-related event metadata.

Preferences

View mode, onboarding and EULA acknowledgement flags, enabled history packs, and app presentation preferences.

Local recovery copy

A local backup/recovery copy of the event store may be kept to protect against accidental persistence failures.

Optional photos

If you choose a custom background photo, the selected image is stored locally as an app-optimized copy.

Widget snapshots

Widget support may use local shared-container snapshots so widgets can display selected countdown information.

Support email you send

If you contact support, your email and the details you include are used to respond to your request.

What does not happen in the current release

  • No account is required for the core experience.
  • No third-party analytics SDK or crash-reporting SDK is used in the current release.
  • The current release does not track users across apps or websites.
  • Apple Ads attribution code exists as a disabled path and is not requested in the current release.
  • The app does not request camera, microphone, contacts, location, HealthKit, or notification permission in the current release.

Photo-library access

Photo-library permission is requested only if you choose to select a custom event background. You can avoid this optional feature and still use the core countdown app.

You can remove selected photos in the app, revoke photo-library access in iOS Settings, or delete the app to remove local app data.

Retention and deletion

Local data

Countdowns, preferences, history-pack choices, and local photo copies remain until you delete them or remove the app.

Support email

Support messages are retained as needed to respond, troubleshoot, and maintain support history.

Website hosting

If this site is hosted on GitHub Pages, the hosting platform may log visitor IP addresses for security and operations. That host-level behavior is separate from the app.
